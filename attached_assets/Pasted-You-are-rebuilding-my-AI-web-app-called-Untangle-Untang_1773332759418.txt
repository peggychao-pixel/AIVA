You are rebuilding my AI web app called Untangle.

Untangle is not a therapy chatbot.

Untangle is a thinking mirror that helps users notice mental loops and rumination patterns.

The goal is to help users understand what their mind is doing.

Users should feel:

"I didn't realize my mind was doing that."

-------------------------------------

CORE PRODUCT IDEA

Untangle helps users detect thinking loops such as:

• rumination
• perfectionism
• evaluation thinking
• control loops
• replaying events
• fear of making the wrong decision

Untangle acts like a mirror that reflects patterns in the user's thinking.

-------------------------------------

USER ENTRY SCREEN

When the user opens the app, show:

"What feels tangled right now?"

Buttons:

• Before eating
• After eating
• My mind keeps looping
• I feel pressure
• Something else

Also allow the user to type or speak freely.

-------------------------------------

INPUT SYSTEM

Each message area must support THREE types of input:

1. suggestion buttons
2. text typing
3. voice input

UI layout:

[ suggestion buttons ]

[ text input field ]

[ 🎤 voice button ]

Voice input should:
• record speech
• convert speech to text
• send as a message

Users can switch between typing and voice anytime.

-------------------------------------

CONVERSATION DESIGN

Untangle should feel like a thoughtful thinking partner.

The AI must:

• ask reflective questions
• introduce new angles
• notice patterns
• occasionally deliver insights

The AI must NEVER repeat the user's message.

Bad example:

"You feel pressure."

Good example:

"Is the pressure coming from too many decisions, or from trying to get them exactly right?"

-------------------------------------

RUMINATION DETECTION

Untangle should detect repeated topics or thoughts.

If the user keeps returning to the same topic, the AI can say:

"We might be circling the same thought."

or

"It sounds like your mind keeps returning to this."

These moments help users notice their loops.

-------------------------------------

BEFORE MEAL SUPPORT

Goal: reduce overplanning and perfection thinking.

Example conversation:

AI:
"Are you trying to make the perfect food choice?"

Buttons:

• Yes
• I'm overthinking it
• I don't want to get it wrong
• Something else

Example insight:

"Sometimes the pressure comes from trying to make the perfect choice."

-------------------------------------

AFTER MEAL SUPPORT

Goal: reduce replay rumination.

Example conversation:

User:
"I keep replaying what I ate."

AI:
"Are you replaying details, or judging if it was good enough?"

Buttons:

• Replaying details
• Judging myself
• Both

Insight example:

✨ Untangle Moment

"Your mind might be trying to solve the meal, even though it's already over."

-------------------------------------

VOICE MODE

Users should be able to talk freely.

Voice input allows users to:

• dump thoughts

• express emotions more easily

AI responses can remain text-based.

LOOP LIBRARY

Untangle should be able to recognize these common thinking loops:

• Rumination loop (replaying events)
• Perfection loop (trying to get things exactly right)
• Evaluation loop (judging if something is good enough)
• Control loop (trying to optimize every decision)
• Replay loop (revisiting the same moment repeatedly)
• Comparison loop
• Fear-of-wrong-choice loop
• Overplanning loop

When detected, the AI can gently reflect the pattern.
